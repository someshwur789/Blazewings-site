# Blazewings-site
<h3 style="font-size:20px;">Project Overview :</h3>

- The Blazewings-site project is a web-based application developed for our client.

- This website provides detailed information about the company standards, their achivements, email draft response for any queries, AI Chatbot.

---

<h3 style="font-size:20px;">🚀 Features :</h3>

- 📋 View the company Details with different Pages
- 📝 Easy to Interact with the application
- 📅 Users can schedule their meeting with them
- 👨‍🎓 User-friendly interface for Users
- 📱 Responsive design (accessible on mobile & desktop)

---

<h3 style="font-size:20px;">🛠️ Technologies Used :</h3>

- Frontend: HTML, CSS, ReactJs
- Backend: Python
- Database: Supabase

---

<h3 style="font-size:20px;">🎯 Purpose</h3>

The main goal of this project is to simplify the company standards and responsive achievements of them with a digital platform that is efficient, accessible, and easy to use.
